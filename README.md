# 💰 Expense Tracker

A simple Python command-line application to record, manage, and analyze daily expenses. The application stores transactions in a JSON file and provides category-wise spending reports.

## Features

- ➕ Add new expenses
- 📋 View all transactions
- 📊 Analyze expenses by category
- 💾 Store data using JSON
- 🗂️ Multiple expense categories
- ❌ Input validation and error handling

## Technologies Used

- Python
- JSON
- File Handling
- Object-Oriented Programming (OOP)

## Installation

1. Clone this repository.
2. Run the project:

```bash
python expense_engine.py
```
